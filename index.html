課程介紹

本課程是一套完整的作業系統學習教材，專為具備基礎程式設計能力（熟悉 C 語言）並希望深入理解作業系統運作原理的學習者設計。課程內容涵蓋從基本的行程管理、記憶體管理、檔案系統到進階的虛擬化技術、分散式系統與雲端運算，並透過大量的 C 語言實作練習與專案開發，幫助學習者建立紮實的作業系統設計與分析能力。最後一週將整合所有知識完成一個實際的 Side Project。

課程特色

  • 16週系統化課程規劃
  • 理論與實作並重教學
  • 完整的 C 語言程式範例
  • 豐富的演算法實作練習
  • 每週配有作業與測驗
  • 實際專案開發經驗

學習時間

  • 建議每週 10-12 小時
  • 理論學習 3-4 小時
  • 程式實作 5-6 小時
  • 複習與測驗 2 小時
  • 期中期末專案額外 15-20 小時

適合對象

  • 具備 C 語言基礎者
  • 資訊工程相關科系學生
  • 準備技術面試的求職者
  • 想深入理解 OS 運作者
  • 系統程式開發者

學習成果

  • 精通作業系統核心概念
  • 掌握系統程式設計技巧
  • 具備並行程式設計能力
  • 理解記憶體管理機制
  • 完成實際系統專案開發

第一階段：作業系統基礎與行程管理 (Week 1-4)

建立作業系統核心概念，深入學習行程的生命週期、狀態轉換，掌握各種 CPU 排程演算法的運作原理與效能分析，並透過 POSIX Threads 學習多執行緒程式設計，為後續課程打下紮實基礎。

第二階段：同步與記憶體管理 (Week 5-7)

深入學習行程同步機制，掌握 Mutex、Semaphore、條件變數等同步工具的使用，解決經典的同步問題。接著學習記憶體配置策略、虛擬記憶體技術與頁面置換演算法，理解現代作業系統如何高效管理記憶體資源。

實作項目：使用 Mutex 保護共享變數、生產者-消費者系統、讀者-寫者系統、死結示範與預防
